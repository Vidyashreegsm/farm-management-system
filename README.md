# 🌾 Farm Management System

A Flask + MySQL based agricultural management system.

---

## 🚀 Features

- Farmer Management
- Crop Management
- Fertilizer Management
- Profit/Loss Analysis
- Dashboard Visualization
- Farmer Detail Tracking
- Search Farmers
- Role-Based Login System
- Responsive UI

---

## 🛠 Technologies Used

- Python Flask
- MySQL
- HTML
- CSS
- Bootstrap
- Jinja2

---

## 📊 Modules

### 👨‍🌾 Farmer Module
- Add Farmer
- Update Farmer
- Delete Farmer
- Farmer Details

### 🌾 Crop Module
- Add Crops
- View Crops
- Profit Calculation

### 🧪 Fertilizer Module
- Add Fertilizers
- Fertilizer Usage Tracking
- Stock Monitoring

### 📈 Dashboard
- Profit/Loss Visualization
- Farmer Statistics
- Fertilizer Statistics
